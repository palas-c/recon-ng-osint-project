# Automated OSINT Reconnaissance using Recon-ng

This project demonstrates passive open-source intelligence (OSINT) gathering using the Recon-ng framework on Kali Linux. The objective is to collect and analyze publicly available intelligence relevant to red-team reconnaissance.

## Project Workflow
1. Recon-ng workspace setup
2. Execution of OSINT reconnaissance modules
3. Review of collected intelligence
4. Documentation of findings

## Tools & Technologies
- Recon-ng
- Kali Linux
- OSINT techniques

## Repository Structure
- screenshots/ : Proof of execution
- reports/ : Reconnaissance report
- commands_used.txt : Commands executed

## Disclaimer
This project was conducted strictly for educational purposes using authorized and publicly available information only.
